# BondKonnect UI Map & Dictionary

This document serves as the primary navigation guide for the BondKonnect AI. Use these routes and terms to direct users within the platform.

## Core Navigation (Dashboard)

| Feature | Route | Description | Key UI Elements |
|---------|-------|-------------|-----------------|
| **Hub / Dashboard** | `/dashboard/apps/hub` | Central overview of market and portfolio. | Market Summary, Recent Activity. |
| **Portfolio Analytics** | `/dashboard/apps/portfolio-analytics` | Professional P&L and performance tracking. | P&L Summary Cards, Portfolio Dropdown, Yield Distribution. |
| **Bond Statistics** | `/dashboard/apps/bond-stats` | Real-time market data for all bonds. | Bond Statistics Table, "Customize View" button, Search bar. |
| **Billing & Payments** | `/dashboard/apps/billing` | Manage subscriptions and funding nodes. | "Payment Methods", "Modify Subscription", "Billing History". |
| **Calculator** | `/dashboard/apps/analysis` | Bond math tools (YTM, Duration). | Calculator input fields, "Calculate" button. |
| **Messages / Chat** | `/dashboard/apps/messages` | Communication with brokers or support. | Chat interface, Intermediary approval. |
| **Research Assistant** | `/dashboard/apps/research-assistant` | AI-powered market research tools. | Analysis reports, Market insights. |
| **Account Settings** | `/dashboard/apps/account` | User profile and security management. | Profile details, Security tab. |

## User Roles & Permissions

- **Individual:** Standard investor access to trading and portfolio tools.
- **Agent / Broker:** Professional access with enhanced quoting and transaction management.
- **Authorized Dealer:** Institutional access focused on high-volume trading and dealer-to-dealer quotes.
- **Admin:** Full system control, including user management (`/dashboard/apps/manage-users`) and activity logs.

## Important UI Terminology

- **Institutional Terminal:** Refers to the professional-grade UI design of BondKonnect.
- **Funding Node:** A saved payment method (Card or Mobile Wallet).
- **Mobile Node:** Specifically refers to M-Pesa or Airtel Money linked accounts.
- **STK Push:** The automated popup on a user's phone to authorize a mobile payment.
- **Bond Issue:** The unique identifier for a bond (e.g., IFB1/2023/17).
- **Valuation Metrics:** Data points like Quoted Yield, Dirty Price, and Coupon.
- **Risk Indicators:** Advanced metrics like Duration, Convexity, and DV01.
